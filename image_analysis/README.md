# Analysing posterior neuruloid patterning by contours from boundary
Analyse neuruloid TBXT intendities within day 2 to obtain intensity trends.

### Folders
```
├── countour_intensities/                    # normalised tbxt intensities
├── example_data/                            # toy data to run notebooks
├── notebooks/
        ├── analysis_annulus.ipynb           # analyse annulus micropatterns
        ├── analysis_TBXT_width.ipynb        # analyse thickness of TBXT ring
        └── isoline_plots.ipynb              # analyse neuruloids with contours
├── organoid_analysis/ 
        ├── geometrical_organoid_analysis.py # functions to analyse neuruloids
        └── utils.py                         
└── tbxt_width_dataframes/                   # tbxt width dataframes
```
