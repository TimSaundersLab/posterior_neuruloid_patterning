# posterior_neuruloid_patterning
![image](https://github.com/user-attachments/assets/7027a988-8d03-47f8-bf0d-bfaada854b69)

Studying pattern emergence in micropatterned posterior neuruloids. Phenomenological modelling of patterning using reaction-diffusion and gene regulatory network. Quantifying pattern emergence of TBXT in posterior neuruloids. 

### Suggested set-up of virtual environment for project
Create virtual environment using conda, with standard libraries, listed in `requirements.txt`.
```bash
conda create -n neuruloids python=3.9.12
conda activate neuruloids
conda config --env --add channels conda-forge
conda install --file requirements.txt
```

Git ignores folder 'data/' in this repo.
